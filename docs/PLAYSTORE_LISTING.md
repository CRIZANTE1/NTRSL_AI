## 6. Assets Visuais Necessários

### Ícone do App (obrigatório)
- **Tamanho:** 512 × 512 px
- **Formato:** PNG (sem transparência)
- **Arquivo atual:** `android/app/src/main/res/mipmap-xxxhdpi/ic_launcher.png`
- ⚠️ Verificar se a versão 512×512 existe — o Play Store exige essa resolução específica

### Capturas de Tela (obrigatório — mínimo 2, máximo 8 por tipo)
Precisa de screenshots das seguintes telas:

| Tela | Arquivo sugerido | Prioridade |
|------|-----------------|-----------|
| Tela principal — Registro do dia (`/home`) | `screenshot_home.png` | Alta |
| Resumo semanal e gráficos (`/dashboard`) | `screenshot_dashboard.png` | Alta |
| Coach de IA (recomendações) | `screenshot_coach.png` | Alta |
| Busca de alimentos (FoodPicker aberto) | `screenshot_food_search.png` | Média |
| Histórico mensal (`/historico`) | `screenshot_historico.png` | Média |
| Tela de login | `screenshot_login.png` | Baixa |

**Especificações técnicas:**
- Resolução mínima: 320 px no lado menor
- Máxima: 3.840 px em qualquer lado
- Proporção: entre 16:9 e 2:1 (portrait ou landscape)
- Formato: JPG ou PNG (sem transparência)
- Tamanho: máx. 8 MB por imagem

### Imagem de Destaque (Feature Graphic) — altamente recomendado
- **Tamanho:** 1.024 × 500 px
- **Formato:** JPG ou PNG
- Deve comunicar a proposta do app visualmente (sem texto obrigatório, mas recomendado)

### Vídeo Promocional (opcional)
- Link do YouTube (não Vimeo)
- Duração ideal: 30–120 segundos
- Formato: 16:9

## 7. Hospedagem da Política de Privacidade

O Play Store **exige** uma URL pública para a política de privacidade. Opções recomendadas:

### Opção A — GitHub Pages (Recomendado — Gratuito)
1. Criar branch `gh-pages` ou pasta `docs/` no repositório
2. Converter `PRIVACY_POLICY.md` para HTML
3. Habilitar GitHub Pages nas configurações do repositório
4. URL resultante: `https://crizante1.github.io/NTRSL_AI/privacy-policy`

### Opção B — Supabase Storage
1. Upload do HTML no bucket público `avatars` ou criar bucket `legal`
2. URL resultante: `https://<project>.supabase.co/storage/v1/object/public/legal/privacy-policy.html`

### Opção C — Gist do GitHub
1. Criar Gist público com o HTML da política
2. Usar serviço como https://roughdraft.io para renderizar como página

**Arquivo HTML pronto para deploy:**  
Veja `docs/privacy-policy.html` (gerado a partir de `PRIVACY_POLICY.md`)
